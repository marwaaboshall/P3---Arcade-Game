How to run:

    1- Open index.html in your browser.
    2- Use the following controllers to control the player:
        -Left Arrow.
        -Right Arrow.
        -Up Arrow.
        -Down Arrow.
    3- Try to avoid the enemies.
    4- The games resets itself when you die.
